# HK School Admissions Assistant

## Live URL
https://www.lsistream.ai/admissions.html

## Concept

Helps mainland Chinese students apply to Hong Kong secondary schools by:

1. **Student Profile** — Fill once, use everywhere
2. **School Search** — Database of HK secondary schools with filtering
3. **Auto-Fill Forms** — Pre-fill PDF application forms
4. **Application Tracker** — Track deadlines, status, notes

## Current Features

- ✅ Student profile creation & save (localStorage)
- ✅ School database (12 sample schools, expandable)
- ✅ Filter by Band, Medium, District
- ✅ School shortlist selection
- ✅ Application tracker with status updates
- ✅ Deadline management
- ✅ Notes for each application
- 🚧 PDF upload & auto-fill (framework ready)

## Technology Stack

- Frontend: HTML, CSS, JavaScript (no backend)
- Storage: localStorage (demo mode)
- PDF: Framework ready for pdf-lib integration
- Deployment: Vercel
